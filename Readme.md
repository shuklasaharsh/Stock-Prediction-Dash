## To run
- Clone
```shell
git clone git@github.com:shuklasaharsh/Stock-Prediction-Dash.git
```
- Install Requirements
```shell
pip install requirements.txt
```
- Run
```shell
streamlit run main.py
```

- This application will predict stock prices for the future.
